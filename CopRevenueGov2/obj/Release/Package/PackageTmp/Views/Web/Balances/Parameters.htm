Your City Account | zip check parameters

The Zip Check Parameter page is a useful tool to protect the user from mis-keying tax payment amounts. Presented for each tax type is statistical information to aid the user in determing an appropriate Limit Amount. When paying the taxes via the Zip Check Payment page, the amount is verified against the limit.
